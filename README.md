# Pluit

> Cloud storage with a clean, opinionated interface I actually want to use every day.

**Live:** [pluit.cloud](https://pluit.cloud)

Most cloud storage UIs look like they were designed by people who hate looking at files. Pluit doesn't. It's a pixel-styled, fast, drag-and-drop vault for your stuff — with an "Ask the Vault" AI layer that lets you search your files in plain English.

## Features

- **Drag-and-drop uploads** with live preview
- **Pixel-styled UI** — opinionated, not boring
- **Ask the Vault** — natural-language search over your files
- **Auth + multi-user** via Supabase
- **File preview modals** for common types

## Built with

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (auth + storage + database)
- Anthropic Claude (Ask the Vault)
- Deployed on Vercel
